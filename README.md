# beyond-study

## SW-CAMP Study
> 1. GitHub을 일상화하기 위해서 기본적인 세팅 (Issue, milestone, 개인 branch 등) 을 마친다.
> 2. 캠프에서 배운 것을 Github 또는 개인 notion, blog에 적고 개인 branch에 커밋한다.
> 3. 개인 branch에서 pull request를 활용해 Issue도 해결해본다.
> 4. 위 과정을 완료하면 main branch에 올린다.
